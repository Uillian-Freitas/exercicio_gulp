# ⚙️ Exercício com Gulp

Projeto desenvolvido para prática de automação de tarefas front-end utilizando Gulp, com compilação de arquivos Sass, processamento de scripts e organização de assets.

## 🚀 Funcionalidades

* Compilação de arquivos `.scss` para CSS
* Organização modular de estilos com Sass
* Processamento de scripts JavaScript
* Estrutura separada entre arquivos fonte (`source`) e arquivos gerados (`build`)
* Automação de tarefas com Gulp

## 🛠️ Tecnologias Utilizadas

* HTML5
* JavaScript
* Sass (SCSS)
* Gulp
* Node.js / npm

## 📂 Estrutura do Projeto

```bash
exercicio_gulp/
├── source/
│   ├── images/
│   ├── scripts/
│   │   └── main.js
│   └── styles/
│       ├── config/
│       │   ├── _reset.scss
│       │   └── _variaveis.scss
│       ├── global.scss
│       └── main.scss
├── build/
├── gulpfile.js
├── package.json
└── package-lock.json
```

## ▶️ Como Executar

1. Instale as dependências:

```bash
npm install
```

2. Execute o Gulp:

```bash
npx gulp
```
